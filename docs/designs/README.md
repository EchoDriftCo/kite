# RecipeVault Design Assets

This directory is for UX/UI design specs, mockups, and style guides.

## Structure

- `color-scheme/` — Color palettes, theme definitions
- `login/` — Login page designs and specs
- `components/` — Component-level design specs
- `mockups/` — Full page mockups and wireframes

## Workflow

1. **Vega** creates design specs here
2. **Kovacs** implements them in the Angular frontend
3. **Deckard** verifies the implementation matches the spec

## Format

Design specs should include:
- Visual mockup or detailed description
- Exact color values (hex/HSL)
- Spacing/sizing values
- Component hierarchy
- Any interaction notes (hover states, transitions, etc.)

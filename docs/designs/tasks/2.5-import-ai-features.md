## Overview

Import and AI-powered recipe features: single image import, multi-image import, AI recipe generator, and recipe mixer. These features make RecipeVault powerful — clean UX is critical.

**Features:**
1. **Recipe Import Dialog** — Upload single image, AI parses, review/edit before save
2. **Multi-Image Import Dialog** — Upload multiple images, AI parses each, batch review
3. **AI Recipe Generator** — Describe desired recipe, AI generates full recipe
4. **Recipe Mixer** — Select 2+ recipes, AI combines/remixes into new recipe

## 1. Recipe Import Dialog

**Trigger:** Click "Import Recipe" button (recipe form, recipe list)  
**Type:** Modal dialog

### Layout

```
┌───────────────────────────────────────────┐
│ Import Recipe                        [X]  │  ← Header
├───────────────────────────────────────────┤
│                                           │
│ Upload a photo of a recipe card, page,   │  ← Instructions
│ or screenshot. AI will extract the       │
│ recipe details.                           │
│                                           │
│ ┌───────────────────────────────────────┐│
│ │                                       ││
│ │        📷 Upload Image                ││  ← Upload button / drop zone
│ │                                       ││
│ │    or drag & drop here                ││
│ │                                       ││
│ └───────────────────────────────────────┘│
│                                           │
│ Accepted formats: JPG, PNG, HEIC         │  ← Helper text
│ Max file size: 10MB                       │
│                                           │
│ [Cancel]                    [Continue]    │  ← Actions (Continue disabled until parsed)
└───────────────────────────────────────────┘
```

### Specs

**Dialog Container:**
- **Max-width:** 560px
- **Background:** `$surface-default`
- **Border-radius:** 16px
- **Shadow:** `$shadow-3`
- **Padding:** 24px

**Title:**
- **Text:** "Import Recipe"
- **Font:** 500 20px, `$text-primary`

**Instructions:**
- **Font:** 400 14px/20px, `$text-secondary`
- **Margin-bottom:** 16px

**Upload Zone:**
- **Border:** 2px dashed `$border-default`
- **Border-radius:** 12px
- **Background:** `rgba(241, 245, 249, 0.05)` (subtle tint)
- **Padding:** 48px 24px
- **Text-align:** center
- **Min-height:** 200px

**Upload Button (inside zone):**
- **Icon:** `upload` (48px, `$accent-teal`)
- **Text:** "Upload Image" (500 16px, `$text-primary`)
- **Secondary text:** "or drag & drop here" (400 14px, `$text-hint`)

**Drag-over State:**
- **Border:** 2px solid `$accent-teal`
- **Background:** `rgba(20, 184, 166, 0.1)`

**Helper Text:**
- **Font:** 400 12px, `$text-hint`
- **Margin-top:** 8px
- **Text:** "Accepted formats: JPG, PNG, HEIC" / "Max file size: 10MB"

### Flow

**Step 1: Upload**
- User clicks "Upload Image" or drags file into zone
- File validation (format, size)
- If invalid: Show error snackbar ("Invalid file format" or "File too large")

**Step 2: Parsing**
- Show progress indicator (indeterminate progress bar + "Parsing recipe...")
- API call to parse image
- Parse time: 3-10 seconds typically

**Step 3: Success**
- Replace dialog content with parsed recipe preview (title, ingredients list preview, instructions preview)
- Show confidence score if available ("Confidence: 92%")
- Show warnings if any ("Could not determine prep time — please verify")
- "Continue" button enabled
- "Edit before saving" option (checkbox)

**Step 4: Continue**
- If "Edit before saving" checked: Navigate to `/recipes/new` with pre-filled form data
- If unchecked: Save recipe immediately, navigate to `/recipes/:id`

### Error Handling

**Parse Failed:**
- Show error message: "Could not identify a recipe in this image. Please try a clearer photo or enter the recipe manually."
- Show "Try Again" button (clears upload, returns to upload state)
- Show "Enter Manually" button (closes dialog, navigates to `/recipes/new`)

**Network Error:**
- Show error: "Network error. Please check your connection and try again."
- "Try Again" button

## 3. AI Recipe Generator

**Route:** `/recipes/generate`  
**Purpose:** Generate a recipe from a natural language prompt

### Layout (Desktop)

```
┌────────┬─────────────────────────────────────────────────────────────┐
│        │  AI Recipe Generator                                        │
│  Nav   ├─────────────────────────────────────────────────────────────┤
│  Menu  │  Describe the recipe you want, and AI will create it for   │
│        │  you. Be as detailed or as vague as you like.               │
│        │                                                              │
│        │  ┌────────────────────────────────────────────────────────┐│
│        │  │ What would you like to cook?                           ││  ← Prompt textarea
│        │  │                                                        ││
│        │  │ e.g., "A quick pasta dish with mushrooms and garlic"  ││
│        │  │                                                        ││
│        │  └────────────────────────────────────────────────────────┘│
│        │                                                              │
│        │  ┌──────────────────────────────────────────────────────┐  │
│        │  │ ⚙️ Options (optional)                                 │  │  ← Options card (expandable)
│        │  │                                                       │  │
│        │  │ Cuisine: [Dropdown: Any ▾]                            │  │
│        │  │ Difficulty: [Easy] [Medium] [Hard]                    │  │
│        │  │ Time: [Dropdown: Any ▾]                               │  │
│        │  │ Dietary: [Vegetarian] [Vegan] [Gluten-Free] ...       │  │
│        │  └──────────────────────────────────────────────────────┘  │
│        │                                                              │
│        │  [Cancel]                          [Generate Recipe]        │
│        │                                                              │
│        │  ──── After generation: ────                                │
│        │                                                              │
│        │  ✨ Generated Recipe                                        │  ← Result card
│        │                                                              │
│        │  Title: [Generated Title]                                   │
│        │  Ingredients: [List preview]                                │
│        │  Instructions: [List preview]                               │
│        │                                                              │
│        │  [Generate Again]  [Edit]  [Save Recipe]                    │
└────────┴─────────────────────────────────────────────────────────────┘
```

### Specs

**Page Title:**
- **Text:** "AI Recipe Generator"
- **Font:** 500 32px/40px (desktop), 500 24px/32px (mobile)
- **Margin-bottom:** 16px

**Instructions:**
- **Font:** 400 16px/24px, `$text-secondary`
- **Margin-bottom:** 24px

**Prompt Textarea:**
- **Component:** `mat-form-field` (outline, textarea)
- **Label:** "What would you like to cook?"
- **Placeholder:** "e.g., \"A quick pasta dish with mushrooms and garlic\""
- **Rows:** 6
- **Required:** true
- **Min-length:** 10 characters
- **Max-length:** 500 characters
- **Character counter:** Show below field (e.g., "245 / 500")

**Options Card (Expandable):**
- **Component:** `mat-expansion-panel`
- **Header:** "⚙️ Options (optional)"
- **Default:** Collapsed
- **Content:**
  - **Cuisine dropdown:** "Any", "Italian", "Mexican", "Chinese", "Indian", "American", "French", "Japanese", "Thai", "Mediterranean"
  - **Difficulty chips:** "Easy", "Medium", "Hard" (single-select)
  - **Time dropdown:** "Any", "< 15 min", "15-30 min", "30-60 min", "1-2 hours", "> 2 hours"
  - **Dietary chips:** "Vegetarian", "Vegan", "Gluten-Free", "Dairy-Free", "Nut-Free", "Low-Carb", "Keto", "Paleo" (multi-select)

**Generate Button:**
- **Style:** Primary button (amber)
- **Icon:** `auto_awesome` (sparkle), 20px
- **Text:** "Generate Recipe"
- **Disabled:** If prompt empty or < 10 characters

### Generation Flow

**Step 1: Loading**
- Disable prompt and options
- Show progress indicator (spinner or linear progress)
- Text: "Generating recipe... This may take 10-30 seconds."
- Cancel button (optional): Cancel API request

**Step 2: Streaming Response (Optional Enhancement)**
- If API supports streaming: Show recipe sections as they generate (title → ingredients → instructions)
- Typewriter effect for instructions (optional, nice-to-have)

**Step 3: Success**
- Show generated recipe in result card
- **Title:** Generated title (editable inline or via Edit button)
- **Ingredients:** List preview (first 5, "... and X more")
- **Instructions:** List preview (first 3 steps, "... and X more")
- **Actions:**
  - "Generate Again" — Clear result, return to prompt (keep prompt text)
  - "Edit" — Open recipe form with pre-filled data (allow editing before save)
  - "Save Recipe" — Save immediately, navigate to recipe detail

### Error Handling

**Generation Failed:**
- Show error message: "Failed to generate recipe. Please try again or adjust your prompt."
- "Try Again" button
- Keep prompt text

**API Rate Limit:**
- Show error: "Too many requests. Please wait a moment and try again."
- Disable Generate button for 30 seconds (countdown timer)

## Responsive Behavior

All dialogs and pages are mobile-first:
- **Mobile:** Single column, full-width buttons, smaller padding
- **Desktop:** Wider dialogs, inline buttons, more padding

## Accessibility

- **Dialog titles:** Clear, descriptive
- **Upload zones:** keyboard-accessible file input (hidden native input triggered by button)
- **Drag & drop:** Not required, always provide click-to-upload alternative
- **Progress indicators:** aria-live="polite" announcements ("Parsing image...")
- **Error messages:** aria-live="assertive", clear error text
- **Recipe picker:** keyboard navigation (Tab, Enter to select)
- **Mix options:** Clear labels, checkbox/chip labels

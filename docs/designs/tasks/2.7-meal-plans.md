### AI Consolidation

**Feature:** Combine duplicate/similar ingredients across recipes

**Example:**
- Recipe 1: "2 cups flour"
- Recipe 2: "1 cup flour"
- **Consolidated:** "3 cups flour"

**Handling Ambiguity:**
- Similar but not identical items (e.g., "cooking oil" + "olive oil") → List separately, AI suggests potential consolidation in UI (optional enhancement)
- Different units (e.g., "1 lb chicken" + "8 oz chicken") → Convert to common unit if possible

**Confidence Indicator (Optional):**
- If AI consolidation uncertain, show subtle indicator (e.g., asterisk, tooltip "Auto-consolidated, verify quantities")

## Overview

Cooking mode is a step-by-step guided view for following recipes while cooking. Clean, simple, hands-free navigation. Large text, touch-friendly buttons, timer integration, voice control (optional).

**Route:** `/recipes/:id/cook`

**Key Features:**
- Full-screen layout (minimal UI chrome)
- Large instruction text (24-32px)
- Step navigation (Previous/Next)
- Progress indicator (Step X of Y)
- Integrated timers (start, pause, resume, cancel)
- Active timers panel
- Voice control toggle (optional)
- Read step aloud (text-to-speech)
- Screen wake lock (keep screen on)
- Exit button

## Voice Control

**Condition:** Optional feature, show toggle button in header

**Commands:**
- **"Next" / "Next step"** → Next step
- **"Previous" / "Previous step"** → Previous step
- **"Read" / "Read step"** → Read instruction aloud
- **"Set timer" / "Start timer"** → Start first detected timer for current step
- **"Pause" / "Pause timer"** → Pause all active timers
- **"Resume" / "Resume timer"** → Resume all paused timers

**Visual Feedback:**
- **Listening:** Show indicator (e.g., pulsing mic icon, or overlay message "Listening...")
- **Command recognized:** Brief flash or checkmark

**Implementation:** Web Speech API (SpeechRecognition)

## Screen Wake Lock

**Feature:** Keep screen on while in cooking mode (prevent auto-sleep)

**Implementation:** Web Wake Lock API
```javascript
if ('wakeLock' in navigator) {
  const wakeLock = await navigator.wakeLock.request('screen');
}
```

**Behavior:**
- Acquire wake lock on enter cooking mode
- Release wake lock on exit cooking mode
- Release wake lock if user navigates away (page visibility change)

## Accessibility

- **Page title:** "Cooking Mode: [Recipe Title] — RecipeVault"
- **Keyboard shortcuts:** Arrow keys (prev/next), Space (next), Escape (exit with confirmation)
- **Voice control:** Optional, announced to screen readers when enabled
- **Read aloud:** Text-to-speech for accessibility (not just convenience)
- **Timer announcements:** Screen reader announces timer start/pause/complete
- **Focus management:** Focus on "Next" button by default, allow Tab navigation
- **Exit confirmation:** If timers active, confirm before exiting ("You have active timers. Are you sure you want to exit?")

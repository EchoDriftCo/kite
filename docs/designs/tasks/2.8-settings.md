## 3. Data Section

**Export Recipes Button:**
- **Component:** Secondary button (outlined, accent color)
- **Icon:** `download` (20px, left)
- **Text:** "Export Recipes (JSON)"
- **Action:** Download JSON file with all user's recipes
- **Margin-bottom:** 16px

**Delete Account Button:**
- **Component:** Secondary button (outlined, warn color)
- **Icon:** `delete` (20px, left)
- **Text:** "Delete Account"
- **Color:** `$error` (warn)
- **Action:** Open delete account confirmation dialog

**Export Flow:**
1. Click "Export Recipes"
2. API call to generate JSON export
3. Download JSON file (e.g., `recipevault-export-YYYY-MM-DD.json`)
4. Success snackbar: "Recipes exported successfully!"

**Delete Account Flow:**
1. Click "Delete Account"
2. Open confirmation dialog:
   - **Title:** "Delete Account?"
   - **Message:** "This will permanently delete your account and all your recipes. This action cannot be undone. Are you sure?"
   - **Input:** Email confirmation field ("Type your email to confirm")
   - **Actions:** "Cancel", "Delete Account" (warn color, disabled until email matches)
3. If confirmed: Delete account, log out, navigate to `/login`
4. Show final snackbar: "Account deleted"

## Change Password Dialog

**Trigger:** Click "Change Password" button

**Type:** Modal dialog

### Layout

```
┌───────────────────────────────────────────┐
│ Change Password                      [X]  │
├───────────────────────────────────────────┤
│                                           │
│ Current Password                          │
│ ┌───────────────────────────────────────┐│
│ │                                       ││
│ └───────────────────────────────────────┘│
│                                           │
│ New Password                              │
│ ┌───────────────────────────────────────┐│
│ │                                       ││
│ └───────────────────────────────────────┘│
│ Must be at least 6 characters             │
│                                           │
│ Confirm New Password                      │
│ ┌───────────────────────────────────────┐│
│ │                                       ││
│ └───────────────────────────────────────┘│
│                                           │
│ [Cancel]              [Change Password]   │
└───────────────────────────────────────────┘
```

### Specs

**Dialog:**
- **Max-width:** 480px
- **Padding:** 24px

**Fields:**
- **Current Password:** Type password, required, show/hide toggle
- **New Password:** Type password, required, min-length 6, show/hide toggle, helper text "Must be at least 6 characters"
- **Confirm New Password:** Type password, required, must match new password

**Validation:**
- **Current password incorrect:** Show error "Current password is incorrect"
- **New password too short:** Show error "Password must be at least 6 characters"
- **Passwords don't match:** Show error "Passwords do not match"

**Actions:**
- **Cancel:** Close dialog
- **Change Password:** Submit, show loading state, on success: close dialog + success snackbar "Password changed successfully!"

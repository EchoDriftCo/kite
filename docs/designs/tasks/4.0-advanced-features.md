## 4.1 Cooking Tracking

### Cooking History (`/cooking-history`)

**Layout:**
```
┌────────┬─────────────────────────────────────────────────────────────┐
│        │  Cooking History                                            │
│  Nav   ├─────────────────────────────────────────────────────────────┤
│  Menu  │  ┌──────────────────────────────────────────────────────┐   │
│        │  │  Spaghetti Carbonara          ★★★★☆           [⋮]  │   │  ← History entry card
│        │  │  Cooked 2 days ago                                   │   │
│        │  │  Note: "Perfect! Used less salt this time."          │   │
│        │  └──────────────────────────────────────────────────────┘   │
│        │  ┌──────────────────────────────────────────────────────┐   │
│        │  │  Apple Pie                    ★★★★★           [⋮]  │   │

│        │  └──────────────────────────────────────────────────────┘   │
└────────┴─────────────────────────────────────────────────────────────┘
```

**Card Specs:**
- Recipe title (500 16px, clickable link to recipe detail)
- Star rating (user's rating for that cook, not recipe overall)
- Date (relative time: "2 days ago", or formatted date if > 1 month)
- Note (if present, italic, 400 14px, `$text-secondary`)
- Overflow menu: Edit Log, Delete Log

## 4.3 Ingredient Search

### Ingredient Search Page (`/ingredient-search`)

**Layout:**
```
┌────────┬─────────────────────────────────────────────────────────────┐
│        │  Search by Ingredients                                      │
│  Nav   ├─────────────────────────────────────────────────────────────┤
│  Menu  │  Enter the ingredients you have. We'll find recipes you can │
│        │  make.                                                       │
│        │                                                              │
│        │  ┌────────────────────────────────────────────────────────┐ │
│        │  │ [chicken] [garlic] [onion] [+ Add Ingredient]         │ │  ← Chip input
│        │  └────────────────────────────────────────────────────────┘ │
│        │                                                              │
│        │  [Search Recipes]                                           │
│        │                                                              │
│        │  ─── Results ───                                            │
│        │  ┌──────────┐ ┌──────────┐ ┌──────────┐                    │
│        │  │ [Image]  │ │ [Image]  │ │ [Image]  │                    │  ← Recipe grid
│        │  │ Title    │ │ Title    │ │ Title    │                    │
│        │  │ ✓ 3/5    │ │ ✓ 4/6    │ │ ✓ 2/4    │                    │  ← Match indicator
│        │  └──────────┘ └──────────┘ └──────────┘                    │     (X of Y ingredients match)
└────────┴─────────────────────────────────────────────────────────────┘
```

**Specs:**
- Ingredient chip input: Multi-select, autocomplete from user's past ingredient usage
- Search button: Query recipes that use selected ingredients
- Results: Recipe grid with match indicator (e.g., "✓ 3/5 ingredients" = 3 of 5 recipe ingredients match)
- Sort options: Most matching first, fewest missing ingredients

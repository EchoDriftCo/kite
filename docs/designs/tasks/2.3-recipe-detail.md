## Overview

Recipe detail is the heart of the app. Users view full recipe info, scale servings, check equipment, start cooking mode, edit, share, and more. This page has the most features of any page — it needs to be clean, scannable, and print-friendly.

**Core Features:**
- Hero image
- Title + star rating
- Description
- Metadata chips (times, servings, public badge)
- Tags
- Personal cooking stats
- Nutrition panel
- Equipment check (owned/missing indicators)
- Grocery delivery links
- Scaling controls (servings input, multiplier buttons)
- Ingredients list (scaled quantities, checkboxes, component links)
- Instructions list (numbered steps)
- Recipe links (components, used-in)
- Actions toolbar (40+ possible actions based on ownership/state)

**Part 1 (this spec):** Core layout, content sections, responsive design  
**Part 2 (next spec):** Actions toolbar, dialogs, complex interactions
